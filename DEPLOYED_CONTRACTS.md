## Deployment Information

**Deployment Date:** November 20, 2025  
**Deployer Address:** `0xf4C3914B127571fDfDdB3B5caCE6a9428DB0183b`  
**Solidity Version:** 0.8.20  
**EVM Version:** Shanghai  
**Pattern:** Storage-as-Transient for Shanghai EVM compatibility

### Technical Details

All DeFi contracts use the **Storage-as-Transient pattern** to emulate Cancun's transient storage on Shanghai EVM. This results in:

- **Gas overhead:** +7-12% compared to native transient storage
- **Full compatibility:** Works on Shanghai EVM without Cancun features
- **Future upgrade path:** Can be optimized when Fushuma upgrades to Cancun EVM

### Configuration Files

Contract addresses are configured in:
- **Frontend:** `src/lib/fumaswap/contracts.ts`
- **Backend:** Environment variables in `.env`
- **Smart Contracts:** `fushuma-contracts/.env`
